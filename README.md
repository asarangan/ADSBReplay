This app will read a GPX track file and send out an ADSB signal to replay the flight track. It has been tested with Avare and AvareX, but in theory it should work with any EFB. On Avare, you need to set the GPS source to "Avare Apps Only" to avoid conflict with your internal GPS. On AvareX, the GPS source should be set to "External". You can load the provided example.gpx file to see how it works. I wrote this app for showing the position of the aircraft on a moving map just as it would have during flight, which I can screen-capture and integrate with cockpit videos. The compiled APK file is under app/release/app-release.apk
